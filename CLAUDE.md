## Project

Educational visualization of unique tic-tac-toe (jogo da velha) board generation. Shows all unique board states organized by turn number (n), where uniqueness accounts for rotational and reflective symmetry. X always goes first.

## Stack

- Vite + TypeScript + React
- DaisyUI + Tailwind CSS

## Commands

Commands will be added as the project is scaffolded. Expected:
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — lint
