# Menace

There are two goals here:

1. A companion UI for an actual menace board. This will tell the human which boxes to update and how to update them (because there are only boxes for canonical states).
2. A simulator. Here you can either play a game against the computer, and watch the menace boxes update, or you can have the computer play against itself. All of the boxes should be tracked with history, as well as the win/loss, so it can all be investigated.
